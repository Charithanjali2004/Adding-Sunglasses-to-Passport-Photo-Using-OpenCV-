# Adding-Sunglasses-to-Passport-Photo-Using-OpenCV-

## Overview

Sturdy Octo Disco is not just a simple overlay tool—it’s a hands-on introduction to the exciting world of computer vision. By combining face detection and image manipulation, this project transforms ordinary passport photos into something fun and visually engaging. It demonstrates how powerful yet accessible image processing can be when using the right tools.

## How It Works

The project uses pre-trained face detection models to identify the position of a face within an image. Once detected, key regions such as the eyes are estimated, and a sunglasses image is resized and aligned accordingly. The overlay is then blended seamlessly with the original image to produce a natural and stylish effect.

## Key Highlights
Automatic Face Detection: No manual adjustments needed—just upload and run.
Smart Overlay Positioning: Ensures sunglasses fit properly regardless of face size.
Lightweight and Fast: Runs efficiently even on basic systems.
Beginner-Friendly Code: Clean and easy-to-understand implementation.
